# v4 Round One Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Seed the empty v4 tier with six skills, each operationalizing one verified, published Karpathy or Cherny idea.

**Architecture:** Each skill is a single `v4/skills/<name>/SKILL.md` designed by the `karpathy-boris-architect` agent (which web-verifies the citation while designing — provenance is a hard gate), then structurally verified with grep checks and gated by a `skill-auditor` run. Two bookkeeping edits close the round: a "Current tools" table in `v4/README.md` and a status tracker in `v4/IDEAS.md`.

**Tech Stack:** Markdown skill files; `karpathy-boris-architect` and `skill-auditor` agents; bash for structural verification.

**Spec:** `docs/superpowers/specs/2026-06-10-v4-round-one-design.md`

**No commits:** this project is intentionally not a git repository (per CLAUDE.md). Every commit step is replaced by a verification step. Do not run `git init`.

**Shared template (all six skills must match):**

```markdown
---
name: <kebab-case-name>
description: Use when <WHEN condition> — <what it does in one clause>
tier: v4
inspiration: "<Originator> — <idea> (<source, venue, date>)"
---

# <Title>

**Not this skill if:** <route to nearest-neighbor skill(s) with tier labels>

<core procedure — concrete, decision-table/checklist style, no fluff>

## Provenance

- **Idea:** <the idea in one or two sentences>
- **Where stated:** <talk/post/interview, venue, date — as verified by web search>
- **How this tool operationalizes it:** <one or two sentences>
```

Optional frontmatter fields when the spec assigns them: `chains-to:`, `pairs-with:`.

**Shared verification commands** (used in every skill task; substitute `<name>`):

```bash
# V1 — frontmatter check: expect 4 lines, including "tier: v4" and a quoted inspiration line
grep -E '^(name|description|tier|inspiration):' v4/skills/<name>/SKILL.md

# V2 — required body sections: expect both a "Not this skill if" line and a "## Provenance" line
grep -nE '^\*\*Not this skill if|^## Provenance' v4/skills/<name>/SKILL.md
```
